# Example of Correspondance between Object Names and its Classes

The key "N" gives the name of an object and the key "Cl" gives the class of the object with this name.

```json
[
	{"N":"0","Cl":"MIT"},
	{"N":"1","Cl":"MIT"},
	{"N":"2","Cl":"MIT"},
	{"N":"3","Cl":"NUC"},
	{"N":"4","Cl":"MIT"},
	{"N":"5","Cl":"CYT"},
	{"N":"6","Cl":"MIT"},
	{"N":"7","Cl":"NUC"},
	{"N":"8","Cl":"MIT"},
	{"N":"9","Cl":"CYT"},
	{"N":"10","Cl":"NUC"}
]
```
